# NLP Research Hub

A structured research hub for studying Natural Language Processing from
foundations to systems, with a focus on models, data, evaluation, and applications.

## Coverage
- NLP foundations and linguistic principles
- Classical and neural models
- Transformer-based architectures
- NLP tasks and benchmarks
- System-level NLP (RAG, pipelines)

## Purpose
- Support academic research
- Enable high-quality teaching materials
- Maintain long-term knowledge base

## Repository Structure

- foundations/ : linguistic and mathematical foundations
- models/      : NLP models by paradigm
- tasks/       : NLP tasks (NER, QA, MT, ...)
- data/        : datasets and preprocessing
- evaluation/  : metrics and benchmarks
- systems/     : RAG, pipelines, deployment
- github-projects/ : curated repositories
- teaching/    : teaching and ESP materials
